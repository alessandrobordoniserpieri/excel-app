# Excel App

## Progetto
App web che replica e migliora un file Excel, con interfaccia grafica moderna.

## Stack
- React 19 + TypeScript + Vite + Tailwind CSS v4
- Libreria `xlsx` per leggere file Excel

## Comandi
- `npm run dev` — avvia il dev server (porta 5173)
- `npm run build` — type-check + build produzione
- `npm run lint` — linting con oxlint
- `npm run preview` — preview build di produzione
