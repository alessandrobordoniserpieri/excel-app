# Excel App

App web che replica e migliora un file Excel, con interfaccia grafica moderna.

## Stack

- **React 19** + **TypeScript** + **Vite** + **Tailwind CSS v4**
- **xlsx** per lettura file Excel

## Sviluppo

```bash
npm install
npm run dev
```

## Ambiente

Il progetto è configurato per **GitHub Codespaces** con tutti i plugin VS Code preinstallati. Apri il repo in un Codespace per avere l'ambiente pronto.

## Comandi

| Comando | Descrizione |
|---------|-------------|
| `npm run dev` | Dev server (porta 5173) |
| `npm run build` | Type-check + build |
| `npm run lint` | Linting |
| `npm run preview` | Preview build |
